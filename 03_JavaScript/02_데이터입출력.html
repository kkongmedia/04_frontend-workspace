자바스크립트에서의 데이터 입/출력

1. 데이터를 출력하는 구문

1) [window.]alert("알람창에 출력할 문구");
2) [window.]console.log("콘솔창에 출력하고 싶은 문구");
3) document.write("화면상에 출력할 문구");
4) 선택한요소.innerHTML | innerText = "요소에 출력할 문구";

1)alert 알림창에 데이터 출력해보기

2)console창에 입력한 데이터 출력하기

웹페이지에 들어가서 개발자도구(f12)를 열명 콘솔창이 보이는데, 그 콘솔차에 이벤트가 발생했을시에 출력해줄 데이터를 입력해보자.

3)document.write("화면상 출력할 문구")

해당 버튼을 클릭할 경우 화면 전체에 데이터 값을 출력하도록 해보기



4) 선택한요소.innerHTML | innerText = "요소에 출력할 문구";

자바 스크립트에서 어떤 요소(element) 안의 값(content) 을 가지고 온다거나
변경하고자 할 때 innerHTML, innerText 라는 속성에 접근

2. 데이터를 입력 받는 구문(변수에 기록 가능)

1) 변수 = [window.]confirm("질문내용");
2) 변수 = [window.]prompt("질문내용");
3) 변수 = 선택한요소.속성(id, className, innerHTML, innerText, .... )
4) 변수 = 선택한input요소.value

1) [window.]confirm("질문내용");

confirm 호출시 "질문내용" 과 "확인/취소" 버튼이 존재하는 알람창 발생
확인 버튼 클릭시 true, 취소 버튼 클릭시 false를 반환


2) [window.]prompt("질문내용");

prompt 호출시 "질문내용" 과 입력할 수 잇는 "텍스트 상자" 와 "확인/취소" 버튼이 보여지는 알람창 발생
확인 버튼 클릭시 텍스트 상자에 입력되어 있는 값이 문자열로 반환(아무리 숫자가 써잇어도 문자열로 반환), 취소버튼 클릭시 null 을 반환(글자 입력을 햇어도 취소를 누르면 무조건 null)



4) 선택한 input요소.value

input을 통해 텍스트 상자에 입력한 값들을 innerHTML, innerText가 아닌 value로 접근해야한다.
