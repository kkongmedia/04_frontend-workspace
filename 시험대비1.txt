1. CSS 파일 방식 및 외부 스타일 연결 방법
1) 내부 스타일방식
-> 스타일 태그 내에 기입하는 방식
2) 인라인 스타일방식
-> 스타일을 부여하고자 하는 요소 내에 스타일 속성을 이용해서 직접접으로 바로 기입하는 방식
3) 외부 스타일방식 :
-> 스타일 정보만을 따로 기술하는 .css 외부 문서를 만들고, head부에 link 태그를 통해 연결시켜주는 방식
<link> 태그 이용해서, 하이퍼링크에 .css 파일 링크 걸기
<link rel="stylesheet" href="">


2. HTML에서 head와 body의 역할
1)<head>
-> 머리부를 나타냄 / <meta>, <title>, <link>, <style>, <script>
-> 해당 문서의 각종 정보와 제목, 설명 및 스크립트, 스타일 시트 내용들 작성
2)<body>
-> 몸체부를 나타냄
-> 화면상에 출력해서 보여주고자 하는 모든 정보/내용 들을 작성하는 부분


3. h태그 어떤 종류가 있는지? (몇개가 있는지)
<h1>~<h6>
h1 : 제일 크고 bold 처리 
2~6 순차적으로 작아짐
7 : 존재하지 않음 / 일반글꼴로 취급됨


4. HTML의 기본 구조
<!DOCTYPE html> : html문서형식 선언
<html> : html 문서의 시작
<head> : 머리부를 나타냄
</head>
<body> : 몸체부를 나타냄
</body>
</html>


5. 각종 태그들.
<s> : 취소선
<mark> : 형광펜
<u> : 밑줄
<em,i> : 기울임체
<b,strong> : 굵게(strong은 스크린리더)
<sub> : 아래첨자
<sup> : 윗첨자
<p> : 문단 영역을 나누는 태그, 줄바꿈하고자 한다면 별도의 태그 작성해야됨(<br>)
<pre> : 시작태그에서부터 종료태그까지 입력된 내용을 그대로 표현하는 태그(줄바꿈, 공백) 모양 등
<ul> : 순서없는 목록 태그
<ol> : 순서있는 목록 태그
<ol type=""> : 숫자값, A, a, i, I
<ol start=""> : 숫자(시작값)
<ol reversed="reversed"> : 역순으로 표기
<img,audio,video> : src="삽입하고자 하는 이미지 경로" / alt="이미지를 설명하는 문구"
<div> : 사각형 박스로 영역지정
<span> : 문장 단위로 영역지정
<iframe> : 웹문서 안에 다른 웹페이지 추가하는 태그
<a> : a태그의 href="" 속성으로 링크 만들 수 있음
<radio> : 오로지 한개만 선택해야 될 때
<checkbox> : 여러개 선택 가능할 경우


6. 블럭요소 / 인라인요소의 차이
블럭요소 : 한줄 단위로 영역을 차지함(줄바꿈이 적용되어 이미 존재하는 태그의 다음 줄에 영역 잡힘)
-> p, pre, div, ....
인라인요소 : content(내용) 영역만 차지하는 요소(줄바꿈이 적용되지 않아 옆으로 영역 잡힘-<br> 직접 작성해야 줄 바꿈됨)
-> b, mark, s, img, span, ....


7. 표태그 작성, 병합(col span, row span)
colspan="" : 열(가로) 합치기
rowspan="" : 행(세로) 합치기


8. 아이디랑 클래스의 차이점
id="" / # : 현재 문서내에 '고유'한 값으로 '하나'만 을 작성
class="" / . : 현재 문서내에 '중복' 된 값 작성가능, '여러개' 작성가능


9. input type 속성
date, submit, text 등
<input> : 
type="text" / 자동으로 한 줄 짜리 텍스트를 입력할 수 있는 텍스트 상자
type="password" / 비밀번호를 입력할 수 있는 텍스트 상자(텍스트가 가려져서 점으로 표시됨)
type="search,url,email,phone" / 겉모습은 일반텍스트 상자와 유사하지만 각각의 정보에 맞게 세분화된 기능 제공
type="submit" / 제출
type="reset" / 초기화
type="button" / 일반버튼
type="color" / 색상 선택
type="file" / 첨부하고자 하는 파일 있을 경우
type="hidden" / 특정값을 입력받지않고, 화면에 노출시키지도 않고, 그치만 넘기고자 하는 값이 있는 경우
type="date" / 연도, 월, 일 까지 입력
month : 연도, 월
week : 해당년도 몇번째 주
time : 오전|오후, 시, 분
datetime-local : 연도,월,일,오전|오후,시,분
